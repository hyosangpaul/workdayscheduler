# 스케줄 생성기

## 현재 진행 상태
1. 스케줄 생성 시스템 완료
2. 스케줄 초기화 및 재생성 완료
3. 엑셀 다운받기 완료
4. 사용방법 업로드 완료
5. 배포 완료
6. 계약직 스케줄 개발 완료
7. 사이드 바 및 라우터로 분리 개발 완료
8. 정규직, 계약직 value 수정 시 showschedule -> false로 스케줄 확인 버튼을 눌러야 보이게 수정

## 추가 개발 예정
1. 날짜 정확히 나오게
2. 주말 여부 적용
3. 윤년 여부 체크
4. 직원 이름 설정
5. 지난 스케줄 저장
